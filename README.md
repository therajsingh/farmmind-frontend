# FarmMind Frontend

This repository contains the frontend dashboard for **FarmMind – Farmer Mind to the Device**, an IoT-based smart farming system.

The application provides a clean and responsive user interface to monitor live sensor data, analyze soil health, and view AI-generated crop recommendations.

---

## 🌱 Features

- Live sensor data visualization
- Historical sensor data charts and tables
- Soil analysis using AI insights
- Crop suitability and recommendation pages
- CSV export of sensor data
- Responsive UI with light/dark mode

---

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript
- CSS
- Recharts (for graphs)

---
