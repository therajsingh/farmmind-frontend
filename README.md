# 🌱 FarmMind Frontend

A modern React-based frontend for **FarmMind**, an IoT-powered Smart Farming platform that helps farmers monitor soil health, visualize sensor data, and receive AI-powered crop and soil recommendations through an intuitive dashboard.

---

## ✨ Features

- 🌾 Modern and Responsive UI
- 📊 Real-time Sensor Dashboard
- 🌡️ Soil Parameter Visualization
- 🧪 Soil Health Analysis
- 🌱 Crop Recommendation System
- 🤖 AI-powered Insights
- 📈 Interactive Charts and Graphs
- 🔄 API Integration with Backend
- 📱 Mobile-Friendly Design
- ⚡ Fast Performance with Vite

---

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/FarmMind-Frontend.git
cd FarmMind-Frontend
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the Development Server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
